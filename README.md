## ✨ Features
- Choice-driven interactive stories
- Multiple AI engines (Gemini & Groq fallback)
- Chapter-based navigation
- Story saving with MongoDB
- Authentication with Firebase
- Profile with last saved stories
- Clean dark UI

## 🛠 Tech Stack
**Frontend**
- React
- Tailwind CSS
- Firebase Auth

**Backend**
- Node.js
- Express
- MongoDB (Mongoose)

**AI**
- Gemini
- Groq (fallback engine)
