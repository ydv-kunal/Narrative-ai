# NarrativeAI 🚀

An AI-powered interactive storytelling platform where user choices shape the story.

<img width="1440" height="757" alt="Screenshot 2026-01-25 at 6 56 02 PM" src="https://github.com/user-attachments/assets/90c3e8f8-65eb-4990-9434-d33d80807394" />



## ✨ Features
- Choice-driven interactive stories
- Multiple AI engines (Gemini & Groq fallback)
- Chapter-based navigation
- Story saving with MongoDB
- Authentication with Firebase
- Profile with last saved stories
- Clean dark UI

## 🛠 Tech Stack
**Frontend**
- React
- Tailwind CSS
- Firebase Auth

**Backend**
- Node.js
- Express
- MongoDB (Mongoose)

**AI**
- Gemini
- Groq (fallback engine)
